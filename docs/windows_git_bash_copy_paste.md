# 윈도우 Git Bash 복사 붙여넣기 (windows git bash copy paste)

## 기본 단축키

| 동작 | 단축키 |
|------|--------|
| 복사 | `Ctrl+Insert` |
| 붙여넣기 | `Shift+Insert` |

## Ctrl+Shift+V 설정 (권장)

Git Bash 창 상단 우클릭 → **Options** → **Keys** 탭에서 **Ctrl+Shift+letter shortcuts** 를 활성화합니다.

설정 후 단축키:

| 동작 | 단축키 |
|------|--------|
| 복사 | `Ctrl+Shift+C` |
| 붙여넣기 | `Ctrl+Shift+V` |

## 마우스로 붙여넣기

우클릭 → **Paste**
